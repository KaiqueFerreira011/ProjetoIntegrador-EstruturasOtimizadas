# Gerenciamento de Estoque em Java

Sistema desenvolvido para controle de mercadorias, focando na eficiência de busca e organização de dados.

## 🛠️ Tecnologias e Conceitos Utilizados
- **Java SE**
- **Java Collections API**: 
  - `HashMap`: Utilizado para garantir buscas de produtos por ID com complexidade O(1).
  - `TreeSet`: Implementado para manter a listagem de estoque sempre ordenada alfabeticamente.
- **Programação Orientada a Objetos (POO)**: Encapsulamento, classes e métodos.

## 🚀 Funcionalidades
- Cadastro de produtos.
- Consulta rápida por código (via Hash).
- Relatório de estoque ordenado automaticamente.

Projeto desenvolvido como parte dos meus estudos em Estruturas de Dados e Programação Orientada a Objetos, aplicando conceitos práticos de manipulação de coleções em Java.
